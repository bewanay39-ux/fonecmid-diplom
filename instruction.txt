# Дипломная работа (1С)

## Состав доработок
- Модуль работы с заявками клиентов
- История статусов заявок
- Модуль расчёта управленческой заработной платы
- Автотестирование с использованием Vanessa Automation

---

## Исходный код
Исходный код доработанной конфигурации представлен в формате **1C:EDT**
и размещён в данном репозитории.

---

## Начальная настройка доработанной конфигурации

1. Назначить пользователю роли:
   - ВКМ_ДиспетчерЗаявок
   - ВКМ_Инженер
   - ВКМ_Руководитель

2. Заполнить справочники:
   - ВКМ_Клиенты
   - ВКМ_Сотрудники

3. Для расчёта управленческой заработной платы заполнить регистр:
   - ВКМ_СтавкиСотрудников

---

## Автотестирование

Для автотестирования используется **Vanessa Automation**.

### Структура автотестов
tests/
├─ features/
│ └─ vkm_requests_full.feature
└─ run_va.bat

markdown
Копировать код

### Запуск автотеста
1. Установить Vanessa Automation (файл `VanessaAutomation.epf`).
2. В файле `tests/run_va.bat` указать:
   - путь к `1cv8.exe`,
   - путь к информационной базе,
   - путь к `VanessaAutomation.epf`.
3. Запустить файл `tests/run_va.bat`.

Результат выполнения сохраняется в файл `tests/va_run.log`.
